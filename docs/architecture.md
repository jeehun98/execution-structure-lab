# Architecture

이 저장소의 구조는 computation structure, preservation semantics, primitive realization, structural composition, hardware evidence, runtime optimization, kernel synthesis라는 여러 계층으로 구성됩니다.

이때 preservation semantics는 하나의 독립 단계라기보다, 전체 시스템을 가로지르는 legality layer로 작동합니다.

이 구조의 목적은 의미, 실행, 최적화 사이의 관계를 명시적으로 드러내는 것입니다.
