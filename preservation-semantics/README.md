# preservation-semantics

이 디렉토리는 어떤 변환이 계산의 의미를 보존하는지, 그리고 그 조건이 무엇인지를 정의합니다.

여기에는 order relaxation, representation change, rematerialization, decomposition, rescaling, bounded approximation 같은 보존 성질이 포함됩니다.

이 의미 보존 규칙은 realization, composition, runtime selection, synthesis 전반의 legality boundary로 작동합니다.
